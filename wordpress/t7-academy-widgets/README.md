# T7 Academy Widgets WordPress Plugin

Complete suite of interactive learning widgets for T7 Academy.

## Installation

1. Copy the `t7-academy-widgets` folder to `/wp-content/plugins/`
2. Go to Plugins in WordPress admin
3. Activate "T7 Academy Widgets"

## Available Widgets

### Certificate Widgets (1-5 Sterne)
- **1-Stern Zertifikat** - Beginner level football skills
- **2-Sterne Zertifikat** - Intermediate level football skills
- **3-Sterne Zertifikat** - Advanced level football skills
- **4-Sterne Zertifikat** - Expert level football skills
- **5-Sterne Zertifikat** - Master level football skills

### Learning Widgets
- **Ginga Basic** - Brazilian Ginga basics
- **Brazilian Dance** - Brazilian dance moves
- **Fortschritt** - Progress tracker
- **Rangliste** - Leaderboard
- **Sterne-Badge** - Star badge display
- **First Touch** - First touch techniques
- **First Touch Air** - First touch in air
- **Expert-Zertifizierung** - Expert certification

## Usage

### Via Shortcode

```
[t7_widget type="1-sterne-zertifikat" player_name="Max" player_email="max@example.com"]
```

### Via Gutenberg Block

1. Add a new block
2. Search for "T7"
3. Select the widget you want
4. Configure options

## Shortcode Parameters

- `type` - Widget type (required)
- `player_name` - Player's name (default: "Spieler")
- `player_email` - Player's email (optional)
